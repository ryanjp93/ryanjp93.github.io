DirectX9 Render Core

C++, DirectX9, Graphics

Programming Language: C++
Tools: DirectX9
Year: 2012
Team Size: Solo
Reason: University

Overview
 This render core is created using C++ with Microsoft's DirectX9 SDK. Its main features are mesh rendering, multiple light sources, a first person camera and particle effects. This is the oldest project on this site and is improved on in every aspect by the DirectX11 Render Core (see Featured Projects).
